V8 JS引擎

  • V8 來自 Google 團隊，可以幫助我們解讀執行 JavaScript 程式碼，使之變為機器語言(0101)。
  • chrome 瀏覽器、Node.js 內部都有 V8，所以在這邊我們都可以使用 JS 來撰寫程式。

Node.js 從 Github 來看

連結

  • deps >> Node.js 裝的插件。
  • src >> 核心程式碼，幫我們編譯 JS
  • lib >> API 文件

Node.js 安裝

連結
下載後可到 cmd 輸入 node -v，若有回應版本號，即表示有下載成功。

Node.js 執行 JS 檔案

在一個資料夾內有一隻 app.js 檔案

如果要用 Node.js 運行，則在終端機打上 Node app.js
