Vue-Note

Vue 的前置了解:

  • Vue 的開發版本有較多的錯誤提示、能使用 Chrome 的 Devtools (類似 js 的 console.log() )
  • Vue Devtools 連結

Vue 的起手式:

  • Vue 的應用程式，一次只能綁定一個 HTML 的元素，故建議綁定 #id 較好
  • 不能做巢狀，只會讀取到最外層有綁定的Vue應用程式

Vue 與 Mvvm:

  • Vue是受到 MVVM的啟發，不完全是MVVM
  • Mvvm
    • M Model 操控的內容 (資料)
    • V View 外觀視覺
    • VM ViewModel 自動綁定的部分
  • Mvvm定義
  • 透過 v-model 來操控資料 View (mvvm的概念)

v-text、v-html的輸出方式:

  • 除了雙括號的方式也可以使用 v-text、v-html
  • 差別是一個對文字 一個可對 html的標籤

v-bind 動態新增 HTML 的屬性

v-for、v-if

  • 就像我們使用 js 的 for迴圈跟 if判斷
  • 補充 Md 的語法不能真的打出雙括號，否則hexo s會報錯
